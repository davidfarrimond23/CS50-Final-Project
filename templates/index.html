Welcome!


Welcome to a cool new mapping Web App
(I hope)


To use it, simply upload a file via Upload on the Nav Bar.

Once the file is uploaded, navigate to Data and select the uploaded file.
Or, try some of our already included files!

Hit "Process"... and let the magic happen.

You'll get a list of placenames. Once done, hit Map it! and be presented with a b-e-a-utiful map!

This site uses the QGIS2WEB plugin for QGIS 3.0, courtesy of Tom Chadwin

For development and testing of the python script behind this app, I have utilised
Project Gutenberg's corpus of free online texts.

Therefore, I recommend utilising texts from this site. This app currently only accepts .TXT files, but more are coming.

The script is not, currently, perfect, and I will be adding an option
to mark entries as a "false positive" soon.

Thank you for using this app!
